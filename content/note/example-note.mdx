# Getting Started with TypeScript

TypeScript is a strongly typed programming language that builds on JavaScript, giving you better tooling at any scale.

## Key Benefits

1. Static typing
2. Object-oriented features
3. IDE support
4. Early error detection

## Basic Example

```typescript
interface User {
  name: string;
  age: number;
}

function greet(user: User) {
  return `Hello, ${user.name}! You are ${user.age} years old.`;
}
```

This is just a basic example to demonstrate TypeScript's type system. The interface defines the shape of our `User` object, and the function uses this type to ensure we're working with valid data.
